# Battlezone98Redux BZN Scanner

### A very simple utilty that scans .BZN files (binary or ASCII) for referenced ODF's, and sorts them by name or custom/stock. Also checks if referenced ODF's are present in same directory as the BZN to help with troubleshooting.
